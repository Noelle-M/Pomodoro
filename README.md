# Pomodoro

[View result](https://noelle-m.github.io/Pomodoro/)

1. Assurez-vous d'avoir Node.js et NPM installés sur votre ordinateur. Si ce n'est pas le cas, vous pouvez les télécharger et les installer à partir de la page officielle de Node.js.

2. Dans le terminal, exécutez la commande <pre>npm install</pre> pour installer toutes les dépendances nécessaires au projet.

3. Une fois l'installation terminée, exécutez la commande <pre>npm start</pre> pour lancer l'application.

4. Le navigateur par défaut devrait automatiquement s'ouvrir et afficher l'application.

Si vous rencontrez des erreurs lors de l'exécution de ces commandes, assurez-vous de lire attentivement les messages d'erreur pour comprendre ce qui ne fonctionne pas. Vous pouvez également vérifier que vous utilisez la bonne version de Node.js en tapant la commande <pre>node -v</pre>
Si vous avez besoin d'aide supplémentaire, n'hésitez pas à consulter la documentation officielle de React.
